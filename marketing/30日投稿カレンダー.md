# Netero — X / Instagram 30日投稿カレンダー

Netero とウェイトリストを広めるための build-in-public 運用プラン。
**芯のメッセージ**: 「意志力はあてにならない。環境は裏切らない。」＝ 全員のタスクが公開で流れるタスクSNS Netero で、焦りを燃料に変える。

---

## 運用の前提

| 項目 | 設定 |
|---|---|
| メイン媒体 | **X（@chmod_777_prod）** — ターゲット（積み上げ勢・個人開発・副業・AIで稼ぐ層）の生息地 |
| サブ媒体 | **Instagram** — ビジュアル訴求。ストーリーズ + フィード。週2〜3本 |
| ウェイトリスト導線 | 《ウェイトリストURL》（例: `https://netero-lp.pages.dev/ai-earn/`。公開URLに差し替え） |
| リンクの置き方（X） | 本文にURLを入れると reach が落ちやすい → **リンクは1stリプ or プロフィール固定**。本文は「リプ欄から」で誘導 |
| リンクの置き方（IG） | プロフィールのリンク（link in bio）に誘導。「プロフのリンクから」 |
| 投稿時間（JST） | 朝 7:00-8:00（朝活層）/ 昼 12:00-12:30 / 夜 21:00-22:30（積み上げ報告のゴールデンタイム）。**夜が本命** |
| エンゲージ運用 | 投稿して終わりにしない。**#今日の積み上げ の他人の投稿に毎日5-10件🔥リプ**。「気配」プロダクトの体現＝最強の営業 |

### コンテンツ・ピラー（7種を回す）

1. **共感（痛み）** — 「一人だと続かない」を言語化
2. **コンセプト** — タスクSNS / 焦りが燃料 / 気配
3. **開発ログ（build-in-public）** — 作ってる過程・意思決定
4. **心理/インサイト** — なぜ他人の気配で人は動くのか
5. **機能紹介** — リーグ / 草グラフ / 公開フィード
6. **ドッグフーディング** — 自分の積み上げ実演
7. **CTA / FOMO** — 創設メンバー・登録の後押し

### 週のリズム（無理なく続く型）

| 曜日 | ピラー |
|---|---|
| 月 | 週初めの宣言・鼓舞 |
| 火 | コンセプト / インサイト |
| 水 | 開発ログ |
| 木 | 機能紹介 or 心理 |
| 金 | ドッグフーディング / 週の振り返り |
| 土 | 差別化 / ストーリー |
| 日 | CTA + ソーシャルプルーフ |

### 画像運用メモ

各日に **画像コンセプト** と **生成プロンプト**（画像生成AI にそのまま渡せる英語プロンプト）を用意。
共通トンマナ: **ダーク基調（黒曜石 #050507）× 残り火オレンジ（#ff4d1c→#ff9a3c）のグラデ、Linear 系、余白広め、日本語は大きく1メッセージ**。
IG は正方形(1080×1080) or 縦(1080×1350)、X は横(1600×900)。ロゴマークは ▲。

- **1日1本を厳守しない**。バズった日は追撃、疲れた日は🔥リプだけでも「気配」は出る
- **数字は正直に**。「◯人登録」は実数を入れる。盛ると build-in-public の信頼が死ぬ
- **引用RTを狙う**。宣言系（Day1/8/15/22）は他人が乗っかりやすい＝拡散の起点
- **スレッド化**：開発ログ（水）と機能紹介（木）はスレッドにして保存性を上げる
- **プロフィール固定**：Day1 か Day30 の強い投稿を固定ツイートに
- **リンクは本文に入れない**：reach 対策。1stリプ or プロフィール
- **ハッシュタグは2〜4個**：盛りすぎると逆効果。媒体で主力タグを絞る
