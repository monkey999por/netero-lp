ARENA — 個人開発

毎日 commit する開発者だけが、 リリースに辿り着く。

Netero は「個人開発」に本気の人間だけのタスクSNS。git log のように、全員の開発タスクが公開で流れる。積み上げを晒せば、手は止まらない。放置リポジトリを、もう作らない。

    WHY IT WORKS

    放置リポジトリは、
    誰にも見られてないから生まれる。

    一人だと「まあ後でいいか」で止まる。締切も、レビュアーも、いないから。
    Netero は開発の締切とレビュアーを、他人の視線で代替する。同じように個人開発してる奴の commit が流れてくる。深夜2時でも、誰かがビルドを回してる。

    全員の main が、公開で伸びていく。マージも、停滞も、全部見える。
    01 / public

    タスクは全部 public

    宣言も進捗も完了も、git log みたいに全員に流れる。private ブランチに逃げられない。

    02 / arena

    個人開発者だけの部屋

    ここにいるのは何かを作ってる人間だけ。同じ夜更かし勢の手の動きが、常に見える。

    03 / streak

    草を、切らすな

    GitHub の草そのまま。毎日の消化が積み上がり、止めた日は全員に見える。連続日数が資産になる。

    netero --config
    tasks.visibility
    100% public
    非公開機能は作らない。見せる前提だから効く
    comments.enabled
    false
    マウントも説教も勧誘も、構造的に不可能
    reactions.allowed
    ["🔥"]
    ネガティブな反応は存在しない
    league.cycle
    7d
    毎週月曜、また同じスタートラインに立つ
    HOW IT WORKS

    4コマンドで、動き出す。

    netero join indie
    アリーナに入る「個人開発」アリーナへ。同じものを作る人間だけの部屋。
    netero task add "認証まわりを実装"
    今日のタスクを切る中身＝あなたの攻め方が、全員に見える。
    netero task done
    こなす。フィードに流れる完了した瞬間、全員のタイムラインに流れて 🔥 が付く。
    netero league
    週末、昇格か降格かペースの近い5〜10人で毎週リーグ戦。月曜、また始まる。
