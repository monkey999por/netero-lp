Netero は「個人開発」に本気の人間だけのタスクSNS。git log のように、全員の開発タスクが公開で流れる。積み上げを晒せば、手は止まらない。放置リポジトリを、もう作らない。

    放置リポジトリは、
    誰にも見られてないから生まれる。

    一人だと「まあ後でいいか」で止まる。締切も、レビュアーも、いないから。
    Netero は開発の締切とレビュアーを、他人の視線で代替する。同じように個人開発してる奴の commit が流れてくる。深夜2時でも、誰かがビルドを回してる。

    01 / public

    タスクは全部 public

    宣言も進捗も完了も、git log みたいに全員に流れる。private ブランチに逃げられない。

    02 / arena

    個人開発者だけの部屋

    ここにいるのは何かを作ってる人間だけ。同じ夜更かし勢の手の動きが、常に見える。

    03 / streak

    草を、切らすな

    GitHub の草そのまま。毎日の消化が積み上がり、止めた日は全員に見える。連続日数が資産になる。

    // how — 使い方

    4コマンドで、動き出す。

    v0.1 — join
    アリーナに入る

    「個人開発」アリーナへ。アカウントなしで即スタート。

    v0.2 — declare
    今日のタスクを切る

    「認証まわりを実装」「LP公開」。中身＝あなたの攻め方が全員に見える。

    v0.3 — ship
    こなす。フィードに流れる

    完了した瞬間、全員のタイムラインに commit が流れ、🔥が付く。

    v1.0 — league
    週次リーグで競う

    消化ペースが近い5〜10人で毎週勝負。昇格か、降格か。

    you — contribution / last 26 weeks · current streak: 26
    while (alive) {
      ship(); // 毎日
    }     DON'T BREAK THE CHAIN.
